# Email Risk Analyzer

## Objective

The Email Risk Analyzer is a Python program that scans email content and identifies potentially unsafe or suspicious messages.

## Features

* Detects suspicious keywords and phrases
* Identifies common phishing patterns
* Flags risky emails
* Provides warning messages for unsafe content

## Technologies Used

* Python 3

## How It Works

1. The user enters email content.
2. The program scans for suspicious keywords.
3. If risky phrases are found, the email is flagged.
4. A warning message is displayed to the user.

## Sample Output

Enter email content: Urgent! Click here to verify your account password.

Suspicious keyword detected: urgent

Suspicious keyword detected: verify your account

Suspicious keyword detected: click here

Suspicious keyword detected: password

Warning: This email may be a phishing or risky email.

## Applications

* Email security awareness
* Phishing detection
* Cybersecurity education
* Safe email communication
